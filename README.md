# Janjan

## Requirements

- Ruby
- Bundler
- Node.js
- PostgreSQL
- Redis
- Google Chrome and ChromeDriver for testing

## Setup

    bin/setup

## Run

  1. Run application server

    bin/rails s

  2. Visit http://lvh.me:3000/debug
